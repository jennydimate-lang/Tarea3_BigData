📊 Análisis de Datos en Batch con Apache Spark
🎯 Descripción del Proyecto

Este proyecto implementa un análisis de datos en batch utilizando Apache Spark (PySpark), con el objetivo de procesar un conjunto de datos de ventas, aplicar transformaciones y generar información útil para la toma de decisiones.

Este desarrollo hace parte de la Tarea 3 – Procesamiento de Datos con Apache Spark del curso de Big Data.

🧠 Problema a Resolver

Analizar un conjunto de datos de ventas para identificar:

Total de ventas por ciudad
Promedio de precios por categoría
Productos más vendidos

Este análisis permite entender el comportamiento comercial y detectar patrones relevantes.

📊 Dataset

El dataset utilizado contiene información de ventas con las siguientes columnas:

fecha
producto
categoria
precio
cantidad
ciudad

⚙️ Tecnologías Utilizadas
Python 3
Apache Spark (PySpark)
VS Code / Terminal
🚀 Instalación y Ejecución
1. Clonar el repositorio
git clone https://github.com/tu-usuario/spark-batch-analysis.git
cd spark-batch-analysis
2. Instalar dependencias
pip install -r requirements.txt
3. Ejecutar el proyecto
python src/batch_analysis.py
